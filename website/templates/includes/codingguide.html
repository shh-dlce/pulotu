General Coding Guide:

Note: you will be automatically redirected to this page upon successful completion of a survey section.

For each item of this coding sheet, a description or excerpt must be provided (under "source") to justify the coding decision.

All sources must be uploaded to the online database – if a source is not available in electronic form, the relevant excerpt should be transcribed or scanned and uploaded. Enter the page numbers of sources used according to the instructions given when coding.

If there are any interesting points or difficulties worth noting, please note them in the "Coder’s Notes" section.

If there is not enough information to answer the question, code the item as "missing data".

If you are able to answer the question but have limited confidence in your answer, check the box 'uncertainty' and explain the reason for your uncertainty in the "Coder's Notes" section.

If a particular variable varies within one culture please code the state that is most representative of the culture as a whole, unless otherwise instructed.

It is not always clear whether a cultural feature should be coded as “absent” or “missing data”. Ethnographers are more likely to talk about the features of a culture that are present, rather than systematically list the features that are absent. For this reason, if there is a fair amount of information related to a feature but no mention of the feature itself, the feature should be coded as “absent” rather than “missing data”. If information regarding the feature is incomplete or ambiguous, use your discretion in deciding whether the item should be coded “absent” or “missing data”. If you have limited confidence in your decision, please mark the item with an asterisk and state the reason for your uncertainty in the “coder’s notes” section.

Chrome / Safari Users: before filling out a survey section page, please add all sources for that section to the database. Click "Edit Sources" in the admin menu to add new sources. Please do not use the 'Add New Source' feature on the survey section page.

Firefox Users: you can add sources as you answer each question.
